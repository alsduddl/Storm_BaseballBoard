# Storm_BaseballBoard
Java + STS3 Storm 야구장 좌석 후기 게시판

### 📍 프로젝트 소개
야구장 좌석 후기를 쉽게 확인하고 직접 후기를 작성하여 다른 사람과 공유할 수 있는 게시판 제공함으로써 더 나은 관람 경험을 할 수 있도록 하고 커뮤니티를 활성하고자 함

###  📌 주요 기능
1) 관리자 : 회원 관리, 게시글 관리
2) 회원 : 회원가입, 로그인, 마이페이지, 게시글 작성, 조회, 수정, 삭제, 좋아요 표시, 댓글 작성
3) 비회원 : 게시글 조회, 인기글 조회, 좌석 검색, 구역 추천 페이지 기능만 가능
- 다른 후기 게시글과의 차별점 : 응원 분위기와 만족도를 점수로 표현하여 한 눈에 파악 가능

### 🔧 개발 환경
- Java 11 
- Apache Tomcat 9.0
- Spring Framework 4.3.25
- Security : Spring Security 4.2.3
- Database : Oracle 18c(xe)
- MyBatis 3.4.6
- JSP, CSS
- IDE : STS3(Spring Tool Suite 3), SQL Developer
